Summary

This project was done in order to study how latitude affects weather as you move closer to the equator. The main tools used in collecting, cleaning, and visualizing data were OpenWeatherMap and Python tools such as MatPlotLib and Pandas.

Pages included on this site are used to show individual visualizations with descriptions as well as side by side comparisons, and finally page displaying the full dataset.
